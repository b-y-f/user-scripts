# 喜马拉雅专辑下载器

一个用于解析并下载喜马拉雅专辑音频的 userscript。

## 功能

- 自动解析专辑内音频 ID
- 自动解析真实音频 URL
- 支持选择音质
- 支持文件名前加序号
- 支持浏览器直接批量下载
- 支持导出 aria2 可用的 `txt` 链接文件

## 使用

1. 安装 userscript 管理器，例如 Tampermonkey。
2. 安装 `xmly-downloader.user.js`。
3. 打开喜马拉雅专辑页，URL 需要类似：

```txt
https://www.ximalaya.com/*/123456
```

4. 点击页面右下角按钮，按顺序执行：

```txt
解析ID -> 解析URL -> 下载 / 导出
```

5. 解析 URL 完成后，可以选择音质，并按需勾选“加序号”。

## 下载

点击“下载”会通过浏览器直接批量下载音频。

如果浏览器开启了“每次下载前询问保存位置”，每个文件都会弹出一次保存框。建议在浏览器下载设置中关闭该选项，并提前设置好默认下载目录。

## 导出

点击“导出”会生成一个 aria2 可用的 txt 文件。

- 如果能从接口解析到专辑名，文件名会使用专辑名。
- 如果解析不到专辑名，默认文件名为 `links.txt`。
- 导出的文件名会跟随当前音质和“加序号”选项。

导出内容格式示例：

```txt
https://example.com/audio.m4a
  out=001.音频标题.m4a
```

使用 aria2 下载：

```sh
aria2c -i links.txt
```

## 注意

- 请先进入具体专辑页再使用脚本。
- 解析和下载速度受喜马拉雅接口限制影响，遇到限流时稍后再试。
- 浏览器直接下载适合少量文件；大量文件建议使用“导出”配合 aria2。
